  ## Description

Application where people can find animals they would like to adopt as their pet. 
A place for people to post animals that need to be adopted.

   [Check it on HEROKU](https://evening-earth-50473.herokuapp.com/)
  
  ![Pet Adopt](./public/images/home.jpg)


  #### For this project the following technologies were used:
  * Nodejs
  * npm dotenv
  * npm express, express-session, express-handlebars
  * npm mysql2
  * npm sequelize
  * npm bcrypt
  * npm handlebars
  * Insomnia 
  * @google-cloud/storage
  * CSS Bootstrap 


  ## Installation instructions

  For dependancies run following comand:

  ```npm i```

  ## Usage information

  To load the database use the following commands:

  ``mysql -u root -p``

  ``source db/schema.sql``
  

  To use this app on localhost - run the following commands:

  ``npm run seed``

  ``npm start``
